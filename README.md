# Band Name Generator

A simple Band Name Generator built using **Node.js**, **Express.js**, **EJS**, and **CSS**.

This project generates a random band name by combining a random adjective and a random noun when the user clicks the button.

## Features

- Generates a random band name
- Built with Express and EJS
- Simple and clean UI
- Dynamic footer year

## Tech Stack

- Node.js
- Express.js
- EJS
- CSS
